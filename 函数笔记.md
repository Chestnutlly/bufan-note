## 函数
- 函数内部可以调用其他函数
- 递归思想不适合很大的数，会造成内存溢出
```
function name(n){
    if(n==1){
        return 1;
    }
    return n*name(n-1);
}
这是求阶乘的的递归思想
```
- `var date = new Date(输入的日期)；`可以把年月日字符串转换为日期类型
- 隐式全局变量，是指在函数里面未加var 的变量在window也可以调用，称为隐式全局变量，但是从严格的意义上讲是错误的
- 函数的名称从本质上来讲就是一个函数体
- 形参就相当于函数的局部变量
- 如果需要封装一个代码，只需要执行一次，以后也不需要调用，那么可以用到自执行函数  例如：
```
     1. ;(function(){
         函数内容
     })();    称为自执行函数
```
## 函数的参数
- 函数体本身也作为参数传递，这种称之为回调函数，在处理异步的情况使用最多。
